SB-1 — Définition canonique d’un objet AXE

Chaque axe doit être défini par les champs suivants (fermés, non redondants) :

id

nom

direction_cardinale (N, NE, E, SE, S, SW, W, NW)

type ∈ {fluvial, commercial, politique, militaire, mixte}

villes_jalons (liste ordonnée)

fonction_historique (1 phrase factuelle)

periode_dominante (approximation large)

invariant_structurel (ce qui reste vrai quelle que soit l’époque)
