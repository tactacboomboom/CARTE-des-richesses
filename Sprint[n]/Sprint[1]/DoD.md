DoD — Definition of Done

Sprint DONE ssi :

Il existe 3 toggles indépendants (Couronnes/Axes/Routes).

Les axes ont :

flèches OU indicateur directionnel,

couleur dépendante du type (fluvial/commercial/politique/mixte),

ne peuvent pas être confondus visuellement avec les routes.

Les couronnes peuvent être masquées sans impacter le reste.

La carte est lisible en zoom 1× :

au moins 1 axe identifiable immédiatement,

les routes n’écrasent pas les axes,

les couronnes restent en arrière-plan.

Aucun changement ne casse pan/zoom/rotation.
